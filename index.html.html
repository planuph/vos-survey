ทำตามขั้นตอนเหล่านี้เพื่อเก็บข้อมูลแบบสอบถามใน Google Sheets โดยอัตโนมัติ

  1. สร้าง Google Spreadsheet ใหม่ใน Google Drive ของคุณ
  2. เปิด Extensions → Apps Script แล้ววางโค้ดด้านล่าง
  3. คลิก Deploy → New deployment → Web app
  4. ตั้งค่า Execute as: "Me" | Who has access: "Anyone"
  5. คัดลอก Web app URL นำมาวางในช่องด้านบน
